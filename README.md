# Project Ambrosia - App Summary

Project Ambrosia makes nutrition information more accessible via quick macronutrient look up and analysis by Gemini 1.5 flash. Users query a specific food which is found in the FatSecrets database, and key macronutrients are displayed across 6 gauges. Gemini then summarizes the nutritional quality of the food based on the macronutrient information. The System Prompt prevents it from drawing unreasonable conclusions or acting as a health professional.
